# Mobile Sales Analysis Dashboard (Power BI)

## Project Overview
This project is an interactive Power BI dashboard built to analyze mobile sales data across different dimensions such as time, city, brand, mobile model, and payment method.

## Key Features
- KPI cards for Total Sales, Quantity, Transactions, and Average Price
- Month-to-Date (MTD) and Same Period Last Year comparison
- Sales analysis by City, Brand, and Mobile Model
- Payment method-wise transaction analysis
- Interactive filters for Year, Month, Mobile Model, and Payment Method

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Business Insights
- Identified top-performing brands and cities
- Observed seasonal trends in mobile sales
- Compared current performance with previous year
